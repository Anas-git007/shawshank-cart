Shopping Cart
------------

Overview:
----------
An online shopping cart is a virtual shopping trolley, where shoppers can put all of their want-to-buy products in, 
review to make adjustments in quantity, product attributes, etc., and remove it before or during the checkout if 
they change their mind. The application also allows users to view the history of their past purchases.

Installation:
-------------
1. Python Packages: Ensure you have the following Python packages installed:
   - pyinstaller
   - pillow

   You can install these packages using pip:

2. Running the Application:
- Execute 'shawshank official.exe' to run the application.

Files:
------
- Shawshank Official.exe: Executable file to run the application.
- project.py: Python code file for the project.
